API Documentation

Getting started

To get started, import the required script and CSS file. The script can be loaded in the head or body.

<link rel='stylesheet' href='path/to/lolreplay.css' />
<script src='path/to/lolreplay.js'></script>

The app can now be initialized with

<script>
  const LR = LOLReplay(
    document.querySelector('#main'),
    { ... data ... },
    { ... options ... }
  );
</script>

Images and other assets must be self-hosted and linked through the various API methods shown below.

Various assets were retrieved from Riot Games resources and/or U.GG. These assets are copyrighted and should not be used in a commercial setting.

I'm hosting a public CDN at cdn.mkn.cx/lolreplay with various assets already, feel free to use them. (e.g., If you want the image to the Mountain Drake, you can access it via cdn.mkn.cx/lolreplay/mountain.png.

Setup

const LR = LOLReplay(
  document.querySelector('#main'),
  LRDataObject,
  LROptionsObject
);

LRDataObject (required)

{
  champions: [ ...ChampionObjects ],
  frames: [ ...FramesObjects ]
}

ChampionObjects (required)

{
  name: string,
  img: string,
  player: string,
  team: int
}

name: required
img: required (URL)
player: required
team: required (0 for blue side, 1 for red side)

FramesObjects (required)

{
  obj: {
    setPit: [string, string] || null,
    setDragon: string || null,
    setTurrets: [ ...int ]
  },
  championIndex: {
    pos: [float, float],
    gold: int,
    score: [int, int, int]
  },
}]

setPit: optional (index 0 is type [e.g., rift], index 1 is the name [e.g., Rift Herald])
setDragon: optional
setTurrets: optional (array of ints indicating turret IDs)
pos: required (x and y coordinates)
gold: required
kda: required (kills, deaths, assists)

LROptionsObject (required)

{
  backgroundImage: string,
  height: int,
  turretImage: string,
  dragons: { str: str }
  turrets: { pos: [int, int], team: int},
  player: CustomPlayerOptions,
  hooks: HookEvents
}

backgroundImage: required (url)
height: required (in px)
turretImage: required (if used, URL)
dragons: required if used, key defines name, value is URL of image
turrets: optional, custom turret locations and teams

CustomPlayerOptions (optional)

{
  autoplay: bool,
  repeat: bool,
  disabled: bool
}

autoplay: optional
repeat: optional
disabled: optional

HookEvents (optional)

{
  onStartToggle: function,
  onChampionClick: function,
  onInitialization: function
}

onStartToggle: no return
onChampionClick: returns ChampionObject clicked
onInitialization: no return
