Basic Initialization

Basic initialization with a map, one player, and Baron Nashor objective.

Hint: Try clicking on Morgana via the map!

Javascript code

Note: Make sure you have a div with the id basic on your webpage.

const basicOptions = {
  backgroundImage: "http://ddragon.leagueoflegends.com/cdn/6.8.1/img/map/map11.png",
  height: 250,
  turretImage: "https://cdn.mkn.cx/lolreplay/turret.png",
  dragons: {
    mountain: "https://cdn.mkn.cx/lolreplay/mountain.png",
  },
  monsters: {
    rift: "https://cdn.mkn.cx/lolreplay/rift.png",
  }
}

const basicData = {
  champions: [{
    name: "Morgana",
    img: "https://static.u.gg/assets/lol/riot_static/11.23.1/img/champion/Morgana.png",
    player: "Tommy",
    team: 0
  }],
  frames: [{
    obj: {
      setPit: ['rift', "Rift Herald"],
    },
    0: {
      pos: [12, 12],
      gold: 50,
      score: [0, 0, 0]
    },
  }]
}

const basic = new LOLReplay(
  document.querySelector("#basic"),
  basicData,
  basicOptions
);

basic.initialize();

Customization

You're able to adjust almost every aspect of the client, whether this is with a custom background, custom monsters, the ability to disable the player, or even alter where turrets are - it's all up to you!

Note: You can automatically start and loop frame view with the player options?

Javascript code

Note: Make sure you have a div with the id customize on your webpage.

const customOptions = {
  backgroundImage: "https://2.bp.blogspot.com/--A1wNZhS868/U7L4xKhbQTI/AAAAAAAAS04/rfzak1JCZFY/s412/srdb3-1.jpg",
  height: 400,
  turretImage: "http://assets.stickpng.com/images/58482d7fcef1014c0b5e4a5a.png",
  dragons: {
    mountain: "https://s3.cointelegraph.com/storage/uploads/view/bad02e8b57a64d349aa5eec318298b4b.png",
  },
  turrets: [
    { pos: [11, 11], team: 0},
    { pos: [17, 17], team: 1}
  ],
  player: {
    repeat: false,
    autoplay: false,
    disabled: true
  }
}

const customData = {
  champions: [{
    name: "Just right",
    img: "https://i.kym-cdn.com/photos/images/newsfeed/001/070/061/d96.jpg",
    player: "Mark",
    team: 0
  }],
  frames: [{
    obj: { spawnTurrets: [0, 1] },
    0: {
      pos: [2.5, 2.5],
      gold: 203910832,
      score: [0, 99, 0]
    },
  }]
}

const customize = new LOLReplay(
  document.querySelector("#customize"),
  customData,
  customOptions
);

customize.initialize();

Developer Callbacks

Developers have the ability to directly hook on callbacks to various functions, these functions include: toggling start/stop, clicking on a champion, on frame updates, and more. You're also able to programmatically control the frame index and various other methods straight via Javascript.

Note: Try clicking on the start/stop toggle or a champion here!

Javascript code

Note: Make sure you have a div with the id dev on your webpage.

const devOptions = {
  backgroundImage: "http://ddragon.leagueoflegends.com/cdn/6.8.1/img/map/map11.png",
  height: 250,
  turretImage: "https://cdn.mkn.cx/lolreplay/turret.png",
  dragons: {
    mountain: "https://cdn.mkn.cx/lolreplay/mountain.png",
  },
  monsters: {
    rift: "https://cdn.mkn.cx/lolreplay/rift.png",
  },
  hooks: {
    onStartToggle: () => alert("Start toggle was called"),
    onChampionClick: (e) => alert("Champion was clicked: " + e.name),
    onInitialization: (e) => console.log("Dev version loaded")
  }
}

const devData = {
  champions: [{
    name: "Morgana",
    img: "https://static.u.gg/assets/lol/riot_static/11.23.1/img/champion/Morgana.png",
    player: "Tommy",
    team: 0
  }],
  frames: [{
    obj: {
      setPit: ['rift', "Rift Herald"],
    },
    0: {
      pos: [12, 12],
      gold: 50,
      score: [0, 0, 0]
    },
  }]
}

const dev = new LOLReplay(
  document.querySelector("#dev"),
  devData,
  devOptions
);

dev.initialize();
